# Introducción a PHP

Ejercicios prácticos de introducción a PHP para desarrollo web del lado del servidor.

## Contenido

| Archivo | Tema |
|---|---|
| 01-hola-mundo.php | Sintaxis básica, `echo`, `print`, `var_dump` |
| 02-variables.php | Variables, constantes (`define`, `const`) |
| 03-tipos-dato.php | Tipos de dato: bool, int, float, string, array |
| 04-operadores.php | Operadores aritméticos |
| 05-comparacion.php | Operadores de comparación y nave espacial `<=>` |
| 06-inc-dec.php | Incremento y decremento |
| 07-string-methods.php | Métodos de strings: `strlen`, `strtoupper`, `str_replace`, concatenación |
| 08-arrays.php | Arreglos indexados: `array_push`, `array_unshift` |
| 09-arrays-assoc.php | Arreglos asociativos (equivalente a objetos en JS) |
| 10-isset-empty.php | Funciones `isset()` y `empty()` |
| 11-funciones-arrays.php | `in_array`, `sort`, `asort`, `ksort` |
| 12-condicionales.php | `if/else`, `else if`, `switch` |
| 13-loops.php | `while`, `do while`, `for`, `foreach` |
| 14-foreach.php | `foreach` con arreglos asociativos anidados |
| 15-funciones.php | Funciones con tipos en parámetros |
| 16-types.php | Tipos de retorno, `strict_types`, nullable types (`?string`) |
| 17-require.php | Modularización con `require` |
| 18-json.php | `json_encode`, `json_decode`, `JSON_UNESCAPED_UNICODE` |
| funciones.php | Funciones auxiliares reutilizables |
